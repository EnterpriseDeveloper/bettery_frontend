bettery tokenomics explained

  • what is BTY?
    An extra reward for event participants, spent by the event Host to engage user community.
  • BTY supply
    BTY is swapped 1:1 from BET, which users earn from events via wins or minting and use to bet.
  • BTY demand
    Businesses, advertisers, and influencers buy BTY to access Bettery community via paid events.
